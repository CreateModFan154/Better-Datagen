# Minecraft Datagen Python Script

A lightweight Python script to automatically generate Minecraft block, item, and blockstate JSONs based on your Java mod source.  
Designed to save hours of manual JSON creation.

---

## Features

- Scans Java source for registry entries automatically.
- Generates:
  - Blockstates
  - Block models
  - Item models
- Detects whether an entry is a block or an item based on texture availability.
- Fully customizable templates for models and blockstates.

---

## Requirements

- Python 3.8+
- Basic understanding of your mod structure and `MODID`
- Place your textures in the appropriate folders:
  - `assets/<MODID>/textures/block/`
  - `assets/<MODID>/textures/item/`

---

## Usage

1. Set `MODID`, `JAVA_SOURCE`, and `OUTPUT_ASSETS` in the script.
2. Place your textures in the corresponding block/item folders.
3. Run the script:

```python datagen.py```

JSON files will be generated automatically in your assets folder.

- Run the script

  - blockstates/iron_pipe.json

  - models/block/iron_pipe.json

  - models/item/iron_pipe.json
