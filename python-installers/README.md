# Python Installers for Better Datagen

This folder contains **official Python installers** for running the Better Datagen script. These installers allow you to quickly set up Python and dependencies so you can run the datagen tools without manually configuring your environment.

---

## Contents

* `python-3.x.x-installer.exe` — Official Python installer for Windows
* `python-3.x.x-macos.pkg` — Official Python installer for macOS
* `python-3.x.x.tar.gz` — Official Python source for Linux/other platforms

> **Note:** These are the official Python distributions. They are **not modified** by us in any way.

---

## Getting Started

1. Run the installer for your platform.
2. Make sure to check **“Add Python to PATH”** if available.
3. Open a terminal/command prompt and verify installation:

```bash
python --version
pip --version
```

4. After Python is installed, navigate to the main Better Datagen repo and follow the instructions in the [README](https://github.com/CreateModFan154/Better-Datagen/tree/main).

---

## Notes

* These installers are provided to simplify setup for users who don’t already have Python installed.
* For advanced users with Python already installed, you can skip these and use your existing Python environment.
